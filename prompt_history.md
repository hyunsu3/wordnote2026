1. 영어 단어 / 뜻 을 단어장 노트를 만들어 줘

- 단어 추가 버튼을 헤더 컴포넌트의 암기 시작 버튼 좌측에 배치해 줘
- 단어 추가 버튼 클릭시, 화면 가운데 새로운 단어와 뜻을 입력하는 창을 열어 줘
- 단어를 추가하는 모달 창 열림
  - 단어 인풋
  - 뜻 인풋
  - 챕터 번호
  - 문항 번호
  - 취소, 저장 버튼
  - 입력 후 엔터를 쳐도 입력되도록하고, 버튼을 여러 번 빠르게 눌러도 중복 입력되지 않도록 해 줘
  - 새롭게 단어가 추가되면 단어 장에 단어와 뜻, 챕터 번호-문항 번호가 표시되도록 한다.

2. 단어 입력 창에 일괄 업로드를 만들어 줘

- \*.csv 쉼표로 구분한 엑셀 파일에 단어와 뜻, 챕터 번호, 문항 번호를 파일로 업로드하면 파싱해서 단어장에 입력해 줘

3. 단어 삭제 기능을 구현해 줘

- 단어에 마우스를 올리게 되면 우측 상단에 휴지통 모양의 버튼으로 나타나게 해 줘
- 삭제 버튼 클릭시 확인을 구하는 모달 창을 열어 줘

4. 단어 수정 기능을 만들어줘

- 단어에 마우스를 올렸을 때 우측 상단에 연필 모양의 아이콘으로 나타나게 해 줘
- 수정 버튼 클릭시 링크 정보를 수정할 수 있는 모달 창을 열어 줘

5. 퀴즈 풀기

- 퀴즈 풀기 누르면 등록된 단어 세트들이 x챕터-x문항 의 이름으로 나열되고, 단어 수 표현
- 단어 세트 선택하면 4지 선다 문제로 영어 단어 - 뜻 또는 뜻 - 영어 단어 의 퀴즈가 출제됨.
- 문제 보여질 때 보기 순서는 랜덤으로 섞이도록 함
- 보기 선택하면 정오답 체크해 보여주고 정답 설명 보여주기
- 정오답 여부를 supabase word_stats 에 저장해 오답 노트 관리
